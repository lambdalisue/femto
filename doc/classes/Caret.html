    Overview

    A textarea caret (selection) management class

    This management class only support W3C selection

    - (Array<Integer>) get()

    Get start and end positions of a caret

    Examples:

    # get current caret position
caret.get() # -> [0, 0]
# change selection range to [1, 2]
textarea.setSelectionRange(1, 2)
# get current caret position
caret.get() # -> [1, 2]

    Returns:

    • (Array<Integer>) — an array which first and second items are start and end positions of a current caret respectively.

    - (Caret) set(s, e)

    Set start and end positions of a caret

    Examples:

    caret.get() # -> [0, 0]
# change caret position to [1, 5]
caret.set(1, 5)
caret.get() # -> [1, 5]
# move caret forward 5 characters
caret.set(5)
caret.get() # -> [6, 10]

    - (Caret) collapse(toStart)

    Move a start position of a caret to its end position or vice versa

    Examples:

    caret.set(2, 5)
caret.collapse()

caret.set(2, 5)
caret.collapse(true)
caret.get() # -> [2, 2]

    Parameters:

    • toStart (Boolean) — if it is true then the end position of the caret will move to its start position

    - (String) text()
    - (Caret) text(text, keepSelection)

    Get or replace a selected text

    Examples:

    textarea.value = "foobar"
caret.text() # -> ""
caret.set(1, 5)
caret.text() # -> "ooba"
caret.text('hogehoge')
textarea.value # -> "fhogehoger"
caret.text() # -> ""
textarea.value = "foobar"
caret.set(1, 5)
caret.text('hogehoge', true)
textarea.value # -> "fhogehoger"
caret.text() # -> "hogehoge"
